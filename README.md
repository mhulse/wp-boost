## Installation

Automated, using `cURL` and `tar` on a unix system:

```bash
$ curl -#L https://github.com/mhulse/wp-stub/tarball/master | tar -xzv --strip-components 1 --exclude=*/{.git*,.editor*,README.*,LICENSE,*.sublime*}
```
